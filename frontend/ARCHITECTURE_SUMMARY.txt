================================================================================
ARCHITECTURE PATTERNS
================================================================================

1. ROUTING
   - React Router v6 with lazy-loaded components
   - Route metadata enables automatic menu generation
   - 8 categories: blockchain, contracts, tokenization, compliance, 
     registries, ai, integration, admin
   - Breadcrumb navigation via NavigationContext

2. STATE MANAGEMENT
   - Redux Toolkit with 5 slices
   - Redux Persist: persists auth, settings, demoApp selections
   - Reselect for memoized selectors
   - Real-time data (liveData) NOT persisted

3. AUTHENTICATION
   - JWT with auto-refresh on 401
   - Token storage in localStorage
   - Credentials included for session fallback
   - Protected routes with useAuth hook

4. API INTEGRATION
   - Service layer abstracts API calls
   - apiClient with JWT interceptor
   - Error handling and retry logic
   - WebSocket with polling fallback

5. UI FRAMEWORKS
   - Ant Design: primary UI framework (51 files)
   - Material-UI: specific components (RWAT Registry)
   - Theme switching (light/dark mode)
   - ConfigProvider for theme management

6. REAL-TIME UPDATES
   - WebSocket service with auto-reconnection
   - Fallback to polling
   - Redux liveData slice for updates
   - useWebSocket hook for components

7. PERFORMANCE
   - Code splitting with lazy loading
   - Vendor bundle separation
   - Memoization with React.memo, useMemo, useCallback
   - Redux Persist for client-side caching
